# 💖 CyberLove

**CyberLove** คือเกมแนว **Visual Novel / Dating Sim**  
ที่ผสมผสานเรื่องราวความรักเข้ากับแนวคิดด้าน **Cyber Security**  
ผู้เล่นจะได้พูดคุย เลือกคำตอบ และสร้างความสัมพันธ์กับตัวละครแต่ละสาย เช่น  
Cryptography, Authentication และ Authorization

---

## 🎮 Features

- 📖 ระบบบทสนทนาแบบ Visual Novel
- 💕 ระบบค่าความสัมพันธ์ (Affection)
- 🎨 UI แนวเกมจีบสาว (Glassmorphism)
- ✨ Animation ด้วย Framer Motion

## 🚀 วิธีใช้งาน (How to Run)

### ติดตั้งโปรแกรมที่จำเป็น
ก่อนเริ่มต้นต้องติดตั้งโปรแกรมดังต่อไปนี้

- **Node.js** (แนะนำเวอร์ชัน 18 ขึ้นไป)
- **npm** (มาพร้อมกับ Node.js)

ตรวจสอบเวอร์ชันที่ติดตั้ง:
```bash
node -v
npm -v
```
Clone โปรเจกต์จาก GitHub :
```bash
git clone https://github.com/laziesv/CyberLove.git
cd CyberLove
```
ติดตั้ง Dependencies :
```bash
npm install
```
รันหน้าเว็บ (Development Mode) :
```bash
npm run dev
```
